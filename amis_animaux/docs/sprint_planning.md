
# Sprint Planning – Amis des Animaux

## Project Overview

Amis des Animaux is a web application designed to connect pet owners who share a passion for animals.
The platform allows users to create profiles, add their pets, discover other pet owners, send match requests, and communicate through a messaging system.

The project was developed using:

* Django (Backend)
* Django Templates (Frontend)
* SQLite (Database)
* HTML/CSS/JavaScript
* Git & GitHub for version control
