
# Retrospective – Amis des Animaux

## What went well

* Clear project architecture using Django apps
* Good separation between backend logic and templates
* Efficient use of Django ORM
* Git version control used throughout the project
* Progressive improvement of the UI and user experience

---

## What could be improved

* Real-time notifications could be implemented
* Chat could use WebSockets for live messaging
* More advanced filtering for profiles
* Improved test coverage

---

## Lessons Learned

* Understanding Django architecture (models, views, templates)
* Managing project structure with multiple apps
* Implementing AJAX requests for dynamic UI
* Handling authentication and user permissions

---

## Future Improvements

* Real-time messaging using Django Channels
* Notification system
* Mobile responsive improvements
* Deployment to a cloud platform
